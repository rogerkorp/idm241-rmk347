Alpha Build

Triggers

When loaded into the website, the user will use a mouse to click on a button.

Rules

  • Script listens for user click.
  • When click is recognized, the script changes the class of the button to one with animation rules.
  • The Button then shrinks down a few pixels, as if it's being pressed down
  • Color of the button begins to fade in.
  • Button then grows to be a bit larger than normal state, but then settles down to normal size – like a recoil.
  • Color fades out, back to the same color as the border.
  • After 400ms, the animation is complete and the script changes the class of the button to a non-existant one – resetting the animation.
