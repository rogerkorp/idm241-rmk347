Settings

Triggers

User clicks on a link.

Rules

    NORMAL MODE TO MENU MODE:
  • When the user hovers over the link, their cursor will turn into a pointer
  • When the user clicks that link, it will trigger an onclick event that:
    • Changes the menu display style from "none" to "flex".
    • Changes the classes for both the menu div and the settings div to ones containing animation rules
  • The background fades into a darker, slightly transparent color
  • The settings div quickly rises from below, stopping at the center of the screen.
  • The user can click various checkboxes to apply different settings

    • MENU MODE TO NORMAL MODE:
  • User hovers over the Finish button. The background color of the button darkens out a bit, and the user's cursor will turn into a pointer
  • When the user clicks the button, it will trigger an onclick event that:
    • Changes the classes for both the menu div and the settings div to ones containing new animation rules
    • Activates a delayed function that will change the menu style from "flex" to "none" after the animations have played out in thier entirety
  • The background fades back into one that's fully transparent
  • The settings div quickly falls beneath the screen.

Loops & Modes
