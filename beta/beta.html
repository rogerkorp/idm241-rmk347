Beta Description

URL

Biome Generator Link

Triggers

User clicks on a small button.

Rules

    NORMAL MODE TO MENU MODE:
  • User hovers over link/button
  • On click, the enu rises slowly from the bottom of the screen, revealing new options for the user.
  • Background darkens so that the user's focus is on the menu
  • Mode switches from "normal" to "menu" to preserve the endstate of the animation.

    • MENU MODE TO NORMAL MODE:
  • User hovers over a cross / exit button
  • User clickes the button
  • Mode switches from "menu" to "normal".
  • The animation reverses.

Feedback

    NORMAL MODE TO MENU MODE:
  • User hovers over link/button

  • User clickes the button
  • The animation reverses.

Loops & Modes

  • Menu Mode allows user to tweak word generation, but doesn't allow the user to generate a word until they exit out.
